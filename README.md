Welcome to **ElevatHub**, a student-led initiative dedicated to equipping aspiring professionals with the right skills, exposure, and opportunities. We organize workshops, hackathons, speaker sessions, and peer-led mentoring programs to bridge the gap between college and career.

---

##  About Us

**ElevatHub** is the official student society formed under MAIT to empower learners through:

-  Collaborations with industry professionals
-  Skill-building workshops & bootcamps
-  Speaker sessions & networking events
-  Placement prep, projects, and internships

##  Setup Instructions

```bash
# Clone the repo
git clone https://github.com/shivamyeshu/Mait_society.git

# Go into the project directory
cd Mait_society

# Install dependencies
npm install

# Run the dev server
npm run dev
```
